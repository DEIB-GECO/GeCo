In the broader context of the GeCo project, which already includes a query language and engine for answering complex biological queries and a large repository of processed genomic data, we also focused on simplifying the use of the produced tools and connecting them within an integrated environment that can be used by biologists and clinicians, even with a limited technological background. GeCo 5.0 represents this spin-off project of GeCo.
