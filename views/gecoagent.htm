During the last two years of GeCo we started the development of GeCoAgent, a fully integrated, user-centred web platform aimed at empowering end-user competences for exploiting GeCo in building human genomics applications. We systematically analysed the design processes conducted using GeCo; we learned that these processes reflect a common interaction pattern, in which a data extraction phase produces an informed definition of the datasets necessary to respond to the specific need of the clinician or biologist, and then a data analysis phase selects the computational procedures (including statistical tools and machine or deep learning libraries) and progressively tunes them so as to build and understand results. Data extraction primarily uses GeCo technology (repositories and tools), whereas data analysis uses classic methods of data science. After both phases, datasets and results can be inspected by means of data visualizations.

Along with this pattern, we designed GeCoAgent as a unique multi-modal tool combining a conversational interface for data extraction and analysis with powerful data visualizations. GeCoAgent uses a conversational agent in the form of a chat box and a dashboard with several objects, progressively built and shown to the user as the interaction goes on. GeCoAgent acts as a soft virtual bioinformatics assistant; the dialogic flow between the user and GeCoAgent web-platform integrates, in a continuous interaction, the use of dialogue to drive actions and of visuals to show results. GeCoAgent directly addresses genomics experts, who often get stuck within a routine of fixed interactions with computing systems, and directly produces streamlined, cleaner and error-free solutions from requirements.
