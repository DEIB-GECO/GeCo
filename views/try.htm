WEB Interface
Includes:
  • web interface where bioinformaticians can browse the datasets of genomic features and biological/clinical metadata and build GMQL queries upon them
  • processed data from ENCODE, Roadmap Epigenomic public sources (open, anonymized data for secondary use)
  • future availability of processed data TCGA and of user-friendly services for biologists
REST APIs
Includes:
  • REST APIs provide programmatic access to GMQL repository and query execution engines
